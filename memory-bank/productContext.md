## Product Context

- **Users**: zonal music directors, rehearsal coordinators, choir members.
- **Use cases**: upload/share rehearsal media, coordinate rehearsal schedules, confirm attendance, push inspirational content, manage zones/groups, process payments or subscriptions.
- **Experience goals**: app should feel like a modern Progressive Web App with instant load, offline resilience, and personalized dashboards per role.
- **Key flows**: authentication, media browsing/player, calendars, group communications, song submissions, notifications (web push + in-app).
