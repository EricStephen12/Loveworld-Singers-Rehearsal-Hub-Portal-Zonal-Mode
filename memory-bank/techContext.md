## Tech Context

- **Frontend**: Next.js 15.5.2 (App Router) with React 19.1, TypeScript, Tailwind CSS v4 config, Headless UI, Radix UI, Lucide icons.
- **Editors**: EditorJS suite and TipTap for rich text/song data editing.
- **Media**: `react-player`, custom audio contexts, Tone.js for sound synthesis, OGL for WebGL visuals.
- **Backend/Services**: Supabase, Firebase, KingsPay payment APIs, custom push notification service, translation route (maybe third-party).
- **Tooling**: ESLint 9, Turbopack dev/build, Lighthouse performance script, Next bundle analyzer, Workbox/next-pwa for PWA SWs.
- **Environments**: multiple Next configs (`next.config.*`) for complex/prod/optimized builds plus custom scripts for cache/media updates.
